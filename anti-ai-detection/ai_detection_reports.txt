#### **1. Surface-Level Artifacts (The "Uncanny Valley" of UI)**

AI coding agents tend to prioritize "looking right" over "functioning correctly."

* **The "Happy Path" Illusion:** The site looks perfect if you click exactly where you are supposed to. However, if you click a button twice rapidly, use the browser's "Back" button unexpectedly, or resize the window aggressively, the UI breaks. Vibe coders rarely prompt for edge-case error handling.
* **Inconsistent Micro-Interactions:** You might see a mix of styling libraries (e.g., a Tailwind button next to a Bootstrap modal) because the AI hallucinated a solution from a different training set for a specific feature.
* **"Lorem Ipsum" Logic:** Placeholder text or generic labels (like "Subtitle goes here") that were hard-coded by the AI and missed by the human prompter.

#### **2. Codebase Fingerprints (If Source is Visible)**

* **Comment Overload:** AI models often over-comment obvious code (e.g., `// Function to add two numbers` above `add(a,b)`).
* **Hallucinated or Redundant Dependencies:** You may find `package.json` files filled with libraries that are installed but never actually used, or five different libraries imported to do date formatting.
* **Cyclomatic Complexity:** AI writes code linearly. It often fails to abstract logic into reusable components, leading to massive, repetitive files (e.g., a 2,000-line React component that should be split into ten smaller ones).
* **Generic Naming Conventions:** Variable names often default to `data`, `item`, `response`, or `result` without specific context, or strictly follow a textbook style (like `handleButtonClick`) that lacks the nuance of a senior engineer's naming conventions.

#### **3. Performance & Security Tells**

* **The "Loading" Bloat:** Vibe-coded sites often ship massive bundles because the AI didn't optimize imports. A simple landing page might load 5MB of JavaScript.
* **Security Hallucinations:** A major tell in 2026 is the presence of "imaginary" security configsâ€”code that *looks* like it sanitizes input but actually does nothing, because the AI mimicked the *structure* of security without the *logic*.
